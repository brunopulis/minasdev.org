# Minas Dev Website

## Installation

```
$ git clone https://github.com/minasdev/minasdev.org.git
$ npm install
```

## CLI

#### Run the Development Mode

```
$ npm run dev
```

#### Run the Production Mode

```
$ npm run start
```

#### Run the tests

```
$ npm run test
```
